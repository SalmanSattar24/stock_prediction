STEP 5: GENERATE YOUR FIRST SIGNAL (5 MINUTES)
================================================================================

Open Python and run:

```python

# Initialize system
system = ComprehensiveSignalSystem()

# Generate signal for a stock
signal = system.generate_integrated_signal('AAPL')

# See the results
if signal:
    print(f"Recommendation: {signal.final_recommendation}")
    print(f"Confidence: {signal.weighted_score:.2f}")
    print(f"Position Size: {signal.position_size:.0f} shares")
    print(f"Stop Loss: ${signal.stop_loss:.2f}")
    print(f"Take Profit: ${signal.take_profit:.2f}")
```

Expected Output:

    Confidence: 0.72

REDDIT API SETUP (DETAILED INSTRUCTIONS)
================================================================================

Step 1: Go to Reddit App Page
   https://www.reddit.com/prefs/apps

Step 2: Click "Create an app" or "Create another app"
   - Name: QuantSignalSystem
   - Type: SELECT "script" (personal use)
   - Description: Stock signal analysis
   - Redirect URI: http://localhost:8080
   - Click: Create app

Step 3: Copy Your Credentials
   You'll see:
   - Client ID (the identifier)
   - Client Secret (long random string)

Step 4: Add to .env File
   REDDIT_CLIENT_ID=your_id_here
   REDDIT_CLIENT_SECRET=your_secret_here
   REDDIT_USER_AGENT=QuantSignalSystem/1.0 by YourRedditUsername

   Replace "YourRedditUsername" with your actual Reddit username!

Step 5: Test It Works
   python config.py --validate

================================================================================
COMMON ISSUES & FIXES
================================================================================

Issue: "FINNHUB_API_KEY not found"
Fix:
   1. Check .env file exists: ls -la .env
   2. Check it has your key: grep FINNHUB .env
   3. Verify format: FINNHUB_API_KEY=abc123 (no quotes, no spaces)

Issue: "Rate limit exceeded"
Fix:
   1. Wait 60 seconds
   2. Check your free tier limits
   3. Reduce screening universe size

Issue: "Reddit authentication failed"
Fix:
   1. Verify Client ID (NOT your password!)
   2. Verify Client Secret is correct
   3. Check Reddit username: REDDIT_USER_AGENT=...by correct_username
   4. Verify app type is "script"

Issue: "Permission denied" running setup scripts
Fix:
   chmod +x setup.sh  (Mac/Linux)
   Then: bash setup.sh
